# 明日方舟资源包处理工具

一个用于处理明日方舟游戏资源包的工具，支持解包和打包功能。

## 功能特点

- 解包Unity资源包(.ab文件)
- 选择性提取资源（支持文件预览）
- 提取纹理和文本资源
- 支持自定义输出目录
- 美观的用户界面
- 实时处理日志
- 进度显示

## 安装

1. 确保已安装Python 3.8或更高版本
2. 克隆或下载此仓库
3. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

## 使用方法

1. 运行程序：
   ```bash
   python src/main.py
   ```

2. 解包资源包：
   - 点击"选择资源包"按钮选择.ab文件
   - 可选择输出目录（默认为当前目录）
   - 点击"扫描资源包"按钮扫描资源包中的文件
   - 在弹出的文件选择对话框中选择要提取的文件（支持图片预览）
   - 点击"开始解包"按钮提取选中的文件

3. 打包资源（开发中）：
   - 点击"选择文件夹"按钮选择要打包的文件夹
   - 点击"开始打包"按钮
